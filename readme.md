> A minimalistic cross-platform clipboard manager

## Dev

```
$ npm install
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for macOS and Windows, using [electron-packager](https://github.com/electron-userland/electron-packager).
